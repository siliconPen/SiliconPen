‚ö° What Is PWM?

PWM stands for Pulse Width Modulation.

Instead of outputting a steady voltage, Arduino:

  • switches the pin ON and OFF very fast

  • controls how long it stays ON vs OFF

This ratio is called the duty cycle.

üß† Think of PWM Like This

Imagine flipping a switch very fast:

  • ON for long time ‚Üí looks bright

  • ON for short time ‚Üí looks dim

Your eyes (and motors) don‚Äôt see the rapid switching ‚Äî
they see the average effect.

That‚Äôs PWM.

üìä Duty Cycle Explained

  • 0% duty cycle ‚Üí always OFF

  • 50% duty cycle ‚Üí ON half the time

  • 100% duty cycle ‚Üí always ON

Arduino expresses this using numbers:

  • 0 ‚Üí 0% (OFF)
  • 128 ‚Üí ~50%
  • 255 ‚Üí 100% (FULL ON)

üîå PWM Pins on Arduino Uno

Not all pins support PWM.

On Arduino Uno, PWM pins are marked with ~:

3, 5, 6, 9, 10, 11

Only these pins can use analogWrite().

üíª Basic PWM Example (LED Fade)

void setup() {
  pinMode(9, OUTPUT);
}

void loop() {
  analogWrite(9, 128); // ~50% brightness
}

Even though the function is called analogWrite(),
it is still digital PWM, not true analog voltage.

üîÑ Fading an LED Smoothly

void setup() {
  pinMode(9, OUTPUT);
}

void loop() {
  for (int i = 0; i <= 255; i++) {
    analogWrite(9, i);
    delay(10);
  }

  for (int i = 255; i >= 0; i--) {
    analogWrite(9, i);
    delay(10);
  }
}

This gradually changes the duty cycle, making the LED fade in and out.

üîÅ Potentiometer ‚Üí PWM Control

Now combine what you learned earlier:

void setup() {
  pinMode(9, OUTPUT);
}

void loop() {
  int potValue = analogRead(A0);
  int pwmValue = map(potValue, 0, 1023, 0, 255);
  analogWrite(9, pwmValue);
}

Turning the potentiometer now directly controls LED brightness.
